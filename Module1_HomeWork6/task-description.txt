1. Создать класс который будет называться Student 
и будет принимать в себя свойства: ("id", "name", "surname", "course", "startYear", "endYear", "selfPayment", "faculty", "ratingPoint")
который при создании экземпляра вы будете передавать параметрами.
Example
let student = new Student(0, "Anton", "Kupchikov", 1, 2019, 2023, false, "JS", 95).

2.Добавить в созданный класс Student метод, который выдаст информацию о студенте.
Строку с инфой создаете как вам угодно. Всю инфу можно не использовать, 
но желательно использовать хотя бы минимум 3 свойства.
Example
student.getFullInfo() /// 'Я - Anton Kupchikov,  студент 1 курса факультетa "JS", рейтинговый балл - 95, форма обучения - бютжет'.

3.Создать метод который выдаст вам место в рейтинге студентов согласно успеваемости. Расчет ведем по свойству ratingPoint.
То есть сортируете studentsArr по убыванию ratingPoint (от большего к меньшему). И ищете свой index + 1 (отсчет индексов с нуля потому что) в полученном массиве.
Читаем Array.protoype.findIndex() - поиск индекса в массиве.
Example
student.getRankedPlace() /// 1

4.Создать новый класс Intern который будет населдовать Student (помни про super(params) и все такое).
У класса Intern должно быть свойство companyName, значение которого вы передадите параметром и все свойства Student.
Example
let intern = new Intern(0, "Anton", "Kupchikov", 1, 2019, 2023, false, "JS", 95, 'Google').

5.Добавить классу Intern метод getFullInternInfo(), который выдаст полную инфу о студенте, как например ( ВО-ВТОРОМ ЗАДАНИИ ) и
добавит к этой информации название компании.
Example
intern.getFullInternInfo() /// 'Я - Anton Kupchikov,  студент 1 курса факультетa "JS", рейтинговый балл - 95, форма обучения - бютжет, интрен комании - Google'.

6. Создать в Intern геттер и сеттер с именем currentCourse (get currentCourse, set currentCourse). Когда вы используете геттер, то вы должны получить 
курс на котором вы учитесь в данный момент ( пробуем new Date().getYear() и смотрим на startYear ). Когда вы используете сеттер,
вы передаете число - курс который вы хотите перезаписать. Логика должна принять это число и сравнить с настоящим годом ( пробуем new Date().getYear() ). 
Потом данный сеттер должен перезаписать startYear и endYear согласно вашим вычислениям.
Example
intern.currentCourse /// 1
intern.currentCourse = 3 

Каждое задание - это просто модификация ваших уже созданных классов.
